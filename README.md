# banking_analysis
data analyst project showcasing banking detials by using python, sql, and power bi

# Overview
This project explores banking data to uncover key insights related to customer behavior, transactions, and financial performance. Using Python for data preprocessing, SQL for data extraction and transformation, and Power BI for visualization, this analysis demonstrates a complete data analytics workflow.

# Project Workflow

1. Data Collection
Collected banking transaction and customer datasets (CSV format or SQL database).

2. Data Cleaning (Python)

Removed duplicates and nulls.

Standardized date and numeric formats.

Performed EDA (Exploratory Data Analysis).

3. Database Integration (SQL)

Designed database schema for storing banking data.

Wrote SQL queries for aggregating account balances, loan performance, and customer demographics.

4. Visualization (Power BI)

Connected Power BI to SQL/Python outputs.

Created KPIs showing customer growth, transaction volume, and loan defaults.

Built interactive dashboards with filters and slicers.
